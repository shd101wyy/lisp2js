### Examples

##### ratio.lisp 
simple ratio data type library  
- print_r
- r+
- r-
- r*
- r/

```lisp
    (r+ 3 4)                  ;; => 7
    (r+ (r/ 1 2) (r/ 1 4))    ;; => 3/4
    (r/ 12 16)                ;; => 3/4
```
